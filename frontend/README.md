# v3s Frontend Demo

This folder contains a minimal Vite + React demo to preview the UI components and theme tokens.

Quick start

1. Install dependencies:

```powershell
cd "C:\Users\Pulki\OneDrive\Desktop\mvpp.projects\V3S\frontend"
npm install
```

2. Run dev server:

```powershell
npm run dev
```

The demo runs at `http://localhost:5173` by default.

Files added in this demo:
- `src/components` — `Panel`, `CursorOverlay` components
- `src/styles` — design tokens and base styles
- `src/AppExample.tsx` — example page showing components
